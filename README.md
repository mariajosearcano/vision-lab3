# Imagenes .tiff
Para usar este repositorio es necesario tener imágenes de formato .tiff , que se encuentran en [este enlace](https://drive.google.com/drive/folders/1eSX42VxHwwbcqxMzaQpd2YyZmvTdhsap)
Después de descargar los archivos comprimidos, se deben poner en la carpeta **imgs** que a su vez está dentro de la carpeta **img**, y descomprimirlos
